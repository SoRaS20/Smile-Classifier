Welcome to the Smile Classifier! This web application uses machine learning to classify images based on facial expressions, determining if a person is smiling or not. A fun yet powerful demonstration of AI in action.

Model Overview

The model used for classification is a Support Vector Machine (SVM), chosen for its efficiency in binary classification tasks. The images are preprocessed by resizing, converting to grayscale, and flattening to feature vectors. The classifier is trained to distinguish between "Smiling" and "Not Smiling" faces with high accuracy.

Tools and Frameworks

This project leverages powerful tools and frameworks to build an efficient, scalable solution:

  • FastAPI: High-performance API framework for modern web applications.
  • scikit-learn: Leading machine learning library for model training and evaluation.
  • MySQL: Robust database for managing the classification data.
  • OpenCV & PIL: Libraries for efficient image preprocessing.
